# vue-awesome-demos

收集封装一些基于vue特殊用法，组件，样例

## Build Setup

```bash
# clone the project
git clone https://github.com/WHYXman/vue-awesome-demos.git

# enter the project directory
cd vue-awesome-demos

# install dependency
cnpm install

# develop
npm run dev
```

This will automatically open http://localhost:9528
